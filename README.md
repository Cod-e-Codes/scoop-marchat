# scoop-marchat

Official [Scoop](https://scoop.sh/) bucket for [marchat](https://github.com/Cod-e-Codes/marchat), maintained by the project. This bucket is separate from Scoop default buckets; add it as shown to install.

## Install

```powershell
scoop bucket add marchat https://github.com/Cod-e-Codes/scoop-marchat
scoop install marchat
```

Canonical manifest template: [marchat `packaging/scoop/`](https://github.com/Cod-e-Codes/marchat/tree/main/packaging/scoop) and [PACKAGING.md](https://github.com/Cod-e-Codes/marchat/blob/main/PACKAGING.md).
